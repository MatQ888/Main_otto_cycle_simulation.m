# Engine Thermodynamic Cycle Simulation (MATLAB)

Simulation of a 4-stroke Otto engine thermodynamic cycle developed for the Aerospace Propulsion course at Universidad Politécnica de Madrid.

## Project Description

This project performs a preliminary design and thermodynamic simulation of an internal combustion engine.

The model evaluates the influence of several parameters on engine performance:

- Compression ratio
- Intake valve closing delay (RCA)
- Exhaust valve opening advance (AAE)
- Ignition advance
- Combustion duration
- Engine speed (RPM)

The simulation computes key performance indicators:

- IMEP
- BMEP
- Engine power
- Brake torque
- Specific fuel consumption
- Volumetric efficiency
- Knock/detonation integral

## Tools

- MATLAB
- Thermodynamic modelling
- Combustion modelling (Wiebe function)
